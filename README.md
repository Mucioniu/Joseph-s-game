# Joseph-s-game
In acest program este creata o lista circulara dublu inlantuita cu santinela. Ca exemplu practic pentru a testa metodele se foloseste 
jocul lui Joseph, care presupun urmatoarele: Se considera n persoane aranjate in cerc. Incepand cu o pozitie particulara p0 se efectueaza 
eliminarea persoanelor din i in i. Executia se termina cand ramane o singura persoana in lista. 

Ca elemente folosim o clasa Info care contine doua atribute varsta si nume. Acestea trebuie date de la tastatura si sunt introduse in 
interiorul listei. O data ce acestea au fost introduse se apeleaza functia jocJoseph care elimina pe rand cate un element pana ramane 
unul singur

Programul a fost creat in Dev-C++.
